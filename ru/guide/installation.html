# Установка

# Зависимости

# Node.js

Требуется Node.js (opens new window) версии 12.0.0 и выше

# TypeScript

Если вы используете TypeScript (opens new window), требуется версия 3.9.0 и выше

# NPM

Установка модуля из пакетного менеджера NPM (opens new window)

$ npm install vk-io

Или

$ yarn add vk-io

# Импорт

# CJS

const { VK } = require('vk-io');

const vk = new VK({
	token: 'токен'
});

// ...

# ESM

import { VK } from 'vk-io';

const vk = new VK({
	token: 'токен'
});

// ...
